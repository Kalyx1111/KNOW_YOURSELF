## 🚀 Quick Start

### Windows (Recommended)

1. Extract the ZIP anywhere.

2. Double-click:

```text
START_KNOW_YOURSELF.bat
```

3. Browser opens automatically.

4. Begin your journey:

- Complete the **100-question assessment**
- Explore your **personality type**
- View **radar charts and facet scores**
- Discover **compatibility insights**
- Take the **Attachment Style Assessment**
